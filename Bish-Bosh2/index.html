Bish-Bosh 2.0

Purpose :

Bish-Bosh lists all numbers between 1 and Loop value, but with some exceptions:

  • If the number is evenly divisible by the first number that should be entered, ‘Bish’ is listed instead of the number.
  • If the number is evenly divisible by the second number that should be entered, ‘Bosh’ is listed instead of the number.
  • If the number is evenly divisible by both the first and the second numbers that should be entered, "Bish-Bosh" is listed instead of the number.
